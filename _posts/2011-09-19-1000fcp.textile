h1. 1000 Functional Connectomes Project


&nbsp;

<img class="alignleft" src="http://fcon_1000.projects.nitrc.org/images/fcp_classic2.png" alt="" width="300" height="237" />

<a href="http://fcon_1000.projects.nitrc.org/">1000 Functional Connectomes Project</a>, led by <a href="http://www.childmind.org/en/directory/staff/mmilham">Michael Milham</a>, M.D., Ph.D., is devoted "to initiating open, prospective data-sharing for the neuroimaging community." <a href="http://lfcd.psych.ac.cn/">Xi-Nian Zuo</a>, Ph.D and <a href="http://neurobureau.org">the Neuro Bureau</a> joined forces to release the <a href="http://db.tt/c8DUupG">processed data</a>, as well as the <a href="https://github.com/openconnectome/1000FCP">code</a> we used to process the data.
