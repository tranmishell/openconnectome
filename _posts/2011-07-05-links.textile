h2. Open Source Software Packages


<ul>
	<li><a href="http://www.nitrc.org/projects/mrcap/">Mr. Cap</a> - Magnetic Resonance Connectome Automated Pipeline, written on top of JIST/MIPAV</li>
	<li><a href="http://fly.mpi-cbg.de/~saalfeld/catmaid/">CATMAID</a> - Collaborative Annotation Toolkit for Massive Amounts of Image Data</li>
	<li><a href="https://sites.google.com/a/brain-connectivity-toolbox.net/bct/Home">BCT</a> - Brain Connectivity Toolbox, a matlab toolbox for analyzing brain-graphs</li>
	<li><a href="http://networkx.lanl.gov/">NetworkX</a> - a python package for computing a variety of graph invariants</li>
	<li><a href="http://www.ilastik.org/">ilastik</a> -  a simple, user-friendly tool for image classification and segmentation</li>
	<li><a href="http://www.ini.uzh.ch/~acardona/trakem2.html">TrakEM2</a> - an ImageJ plugin for connectome annotation and image processing</li>
	<li><a href="http://connectomes.utah.edu/">Viking Annotation System</a> - a multi-user web-based collaborative management system for volumes</li>
	<li><a href="http://www.fmrib.ox.ac.uk/fsl/">FSL</a> - a comprehensive library of analysis tools for FMRI, MRI and DTI brain imaging data</li>
	<li><a href="http://afni.nimh.nih.gov/afni">AFNI</a> -  a set of C programs for processing, analyzing, and displaying functional MRI (FMRI) data</li>
	<li><a href="http://nipy.sourceforge.net/nipype/">NiPype</a> - a Python project that provides a uniform interface to existing neuroimaging software</li>
</ul>
