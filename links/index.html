Open Source Software Packages

  • Mr. Cap – Magnetic Resonance Connectome Automated Pipeline, written on top of JIST/MIPAV
  • CATMAID – Collaborative Annotation Toolkit for Massive Amounts of Image Data
  • BCT – Brain Connectivity Toolbox, a matlab toolbox for analyzing brain-graphs
  • NetworkX – a python package for computing a variety of graph invariants
  • ilastik – a simple, user-friendly tool for image classification and segmentation
  • TrakEM2 – an ImageJ plugin for connectome annotation and image processing
  • Viking Annotation System – a multi-user web-based collaborative management system for volumes
  • FSL – a comprehensive library of analysis tools for FMRI, MRI and DTI brain imaging data
  • AFNI – a set of C programs for processing, analyzing, and displaying functional MRI (FMRI) data
  • NiPype – a Python project that provides a uniform interface to existing neuroimaging software
  • Knossos – a software tool for the visualization and annotation of 3D image data

Other Brain-Graph Projects

  • 1000 Functional Connectomes Project – ’nuff said
  • ADHD200 – a grassroots initiative, dedicated to accelerating our understanding of the neural basis of ADHD
  • Retina Connectome – a large open dataset containing some mouse retina with associated open tools
  • human connectome project – NIH’s plan to get human MR connectomes
  • mouse connectome project - aims to create a 3D digital connectome atlas of the C57Black/6J mouse brain
  • brain architecture project – human, mouse, and other species brain architecture and structural organization.
